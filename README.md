## 使用说明

```bash
# 新 pdf 由 dvc 管理
make dvc-add
# 推到阿里云 OSS 备份 pdf 原文
dvc push
# 更新 README 中的 pdf list
make run_pipeline
```

更多用法见：[详细使用说明](docs/how-to-use.md)
